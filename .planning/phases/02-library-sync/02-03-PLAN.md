  <action>
    1. Add APScheduler to `requirements.txt`:
       ```
       apscheduler>=3.11,<4.0
       ```
       Add it after the existing dependencies. Do NOT use APScheduler 4.x (still alpha).

    2. Create `app/services/sync_scheduler.py`:

       **Module-level singleton:**
       ```python
       from apscheduler.schedulers.asyncio import AsyncIOScheduler
       from apscheduler.triggers.interval import IntervalTrigger
       import asyncio

       _scheduler: AsyncIOScheduler | None = None
       ```

       **`get_scheduler() -> AsyncIOScheduler`**: Create singleton if None, return it.

       **`schedule_sync(interval_hours: int)`** (per D-02):
       - Get scheduler
       - Remove existing job "library_sync" if it exists (use `scheduler.get_job("library_sync")`)
       - Add new job:
         ```python
         scheduler.add_job(
             _trigger_sync,
             trigger=IntervalTrigger(hours=interval_hours),
             id="library_sync",
             replace_existing=True,
             name=f"Library sync every {interval_hours}h",
         )
         ```

       **`async def _trigger_sync()`**: Wrapper that calls `asyncio.create_task(run_sync())` -- this is the job function APScheduler calls.

       **`async def start_scheduler()`** (per D-03):
       - Get scheduler and start it
       - Load sync interval from settings: query ServiceConfig for "plex", check `extra_config` for `sync_interval_hours` key. Default to 24 if not set.
       - If interval found: call `schedule_sync(interval_hours)`
       - Auto-sync on startup: check if Plex is configured AND `get_last_sync_info(session)["last_sync_completed"]` is None. If so, launch initial sync: `asyncio.create_task(run_sync())`

       **`async def stop_scheduler()`**: Call `scheduler.shutdown(wait=False)` if scheduler exists.

       **`def update_sync_schedule(interval_hours: int)`** (per D-04):
       - If scheduler is running: call `schedule_sync(interval_hours)`
       - This is called from the settings save endpoint when user changes interval

       - Test start_scheduler triggers auto-sync when no previous sync exists
       - Test stop_scheduler shuts down without error
       - Mock run_sync to avoid actual Plex calls
