<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Plex file path -> container filesystem | Plex-reported paths are remapped and used to read files from /music mount |
| Audio file -> Essentia | Arbitrary audio files processed by C++ library |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Tampering | remap_plex_path | mitigate | Reject paths containing ".." to prevent path traversal outside /music. Validate remapped path starts with container_mount prefix. |
| T-03-02 | Denial of Service | extract_features | mitigate | Per-track timeout (30s) in analysis_service (Plan 02). Skip files > 100MB. Wrap in try/except so one bad file cannot stop batch. |
| T-03-03 | Information Disclosure | analysis error logs | mitigate | Do not include full file paths in error responses to API. Log full paths server-side only. |
</threat_model>
